# Слова, часто используемые в CSS-классах

`img` — картинка
`icon` — иконка
`logo` — логотип

`list` — список
`item` — элемент списка

`header` — шапка
`footer` — подвал

`wrapper` — обёртка
`inner` — внутренний элемент или страница

`content` — содержимое элемента
`text` — текстовый контент
`title` — заголовок
